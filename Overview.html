Ritchie Blackmore Ancestor Database

This is a repository of self-contained HTML files containing direct ancestors of guitarist Ritchie Blackmore.

Disclaimer

Ancestor data was sourced from the website FamilySearch.org on the dates indicated in the table below and as such is subject to error and change. The data on the Family Search website is frequently updated by volunteers who transcribe old handwritten documents that may be innacurate.

Rationale

The purpose in creating these files is twofold: preservation of data offline and ease and flexibility of searching. There is a search tool on the website mentioned above, but it has limited usefulness. The search tool built into the data base files below allow one to quickly find all ancestors with a given title such as King or name such as Blackmore, or all ancestors from a given city such as London or country such as Wales. I created this software for my own use but decided to apply it to Ritchie Blacmore's ancestry after discovering that my favorite guitarist is also my father's distant cousin.

Usage and downloading

The HTML database files in the table below can be opened in a browser and used online simply by clicking on one of the links in the table. Or, each one can be downloaded to your local computer by right-clicking on a link and selecting the option Save link as... Once downloaded, HTML files can be opened in a browser even when you are offline. Alternatively, all HTML files including this overview page can be downloaded immediately as a zip file by clicking here.

Features

  • Self-contained ancestry database usable online or offline, requiring only a browser; no installation needed.
  • Permanent digital record of ancestors even if source website becomes unavailable.
  • Efficient and flexible searching by name, place or date; wildcards may be used.
  • Data includes links to ancestors' source records on FamilySearch.org.
  • Data may be exported in CSV format for use with spreadsheet or database software.
  • Data includes relationship chains leading from Richard Hugh Blackmore to every ancestor in the database.
